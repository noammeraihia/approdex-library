# $\textbf{Appro}\textit{Dex}\text{ library}$
The library of math courses on which the [ApproDex website](https://github.com/noammeraihia/approdex) is based.

## The _sections_ folder
The [sections](sections/) folder contains all the actual data as $\LaTeX$ files. Each file is related to a _chapter_ and is named after it according to the [section_names.txt](section_names.txt).
- _**section**_ refers to to a chapter (e.g., "Variables aléatoires indépendantes")
- _**subsection**_ to a title (e.g., "I.", "VI.", ...)
- _**subsubsection**_ to an actual theorem, formula, proposition, definition...

_subtitles (like "I.1" or "V.3") are not documented as this would complicate matters._ 

## The _bind.tex_ file
This is simply a file gathering all data from the [sections](sections/) folder in one pdf, to be used as an alternative the website [ApproDex website](https://github.com/noammeraihia/approdex) (that is still in development).

## Also
Some _proofs_ may be included in the future but for now only actual course is being documented as _proofs_ are different from one teacher to another.
